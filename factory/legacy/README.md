# Factory Legacy (Deprecated)

Эта папка содержит устаревший код Factory OS версии 1, который **не используется в production**.

## 📁 Файлы

### Frontend
- **`frontend_v1.html`** — оригинальный одностраничный фронтенд (5310 строк)
  - Все компоненты в одном файле
  - Встроенные CSS и JavaScript
  - Использовался до рефакторинга

### Backend (Legacy)
- **`orchestrator_v1.py`** — старая версия оркестратора
- **`agents_v1.py`** — старые агенты
- **`dashboard_v1.py`** — альтернативный dashboard (stdlib HTTP server)
- **`schema_v1.sql`** — старая схема БД
- **`transitions_v1.sql`** — старые FSM переходы

## ⚠️ Важно

Эти файлы **не импортируются** из основного пакета `factory/` и **не участвуют в CI**.

Для запуска используйте новый фронтенд:
- `factory-os-refactored.html` в корне проекта
- Или через API server: `python -m factory.api_server` (порт 8000)

## 📊 Отличия от новой версии

| Аспект | Legacy v1 | Новый (refactored) |
|--------|-----------|-------------------|
| Архитектура | Монолит | Модульная (ES6) |
| CSS | Встроенный | `/static/css/factory.css` |
| JavaScript | Встроенный | `/static/js/components/*` |
| State | Глобальные переменные | Store с pub/sub |
| API calls | Inline функции | Централизованный client |
| Coverage | 100% | ~75% (в процессе) |
